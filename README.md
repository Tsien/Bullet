# Bullet
A shell script for setting up workspace on a brand new OS

## Set up a new computer(Mac OS):

* iTerm + zsh
* Google Chrome
* Alfred
* Sublime Text
    * plugins:
        * SublimeLinter
        * SublimeCodeIntel
        * Emmet
        * MarkdownEditing
        * MarkdownPreview
        * DocBlockr
* Git
* Franz
